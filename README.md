# Информационная система магазина обуви (ООО «Обувь»)

Данный проект представляет собой информационную систему для магазина по продаже обуви. Разработан в рамках выполнения задания демонстрационного экзамена.

## Описание проекта

Приложение разделено на модули, доступ к которым разграничен в зависимости от роли пользователя. Неавторизованный пользователь может войти в систему как гость или авторизоваться, используя учетные данные из базы данных.

### Роли пользователей и функционал

1.  **Гость**
    - Доступен экран просмотра списка товаров без применения фильтрации, сортировки и поиска.

2.  **Авторизованный клиент**
    - Просмотр списка товаров (без фильтрации, сортировки и поиска).

3.  **Менеджер**
    - Просмотр списка товаров с возможностью **фильтрации**, **сортировки** и **поиска**.
    - Просмотр списка заказов.

4.  **Администратор**
    - Полный доступ к управлению товарами: просмотр (с фильтрацией, сортировкой и поиском), добавление, редактирование и удаление.
    - Полный доступ к управлению заказами: просмотр, добавление, редактирование и удаление.

### Интерфейс

- **Окно входа**: первое окно, которое видит пользователь. Содержит поля для ввода логина и пароля, а также кнопку для входа в режиме "Гость".
- После успешной авторизации открывается доступ к модулям, соответствующим роли пользователя.

## Технологии

- Язык программирования: C#
- База данных: PostgreSQL
- Интерфейс: Avalonia
